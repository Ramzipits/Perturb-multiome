# Code Overview

This repository contains illustrative scripts outlining the workflow structure as a conceptual framework.

1. **1_GBCcount_CBC**  \
   Example code for quantifying GBC library counts and processing cell barcodes.  \
\
2. **2_GBC_cellID**  \
   Linking single-cell perturbation identities with corresponding GBC information.  \
\
3. **3_brain_blood_list**  \
   Example list of aging-associated lncRNAs derived from brain and blood datasets.  \
\
4. **4_K562_process_integration**  \
   Perturb-multiome data processing in K562 cells, also filtering, and dimensionality reduction.  \
\
5. **5_chromVAR_dimention_reduction**  \
   Illustrative dimensionality reduction structure.\
\
6. **6_chromVAR_pertcell_filter_DEG**  \
   Perturbed cell filtering and DEG calling structure.  \
\
7. **7_ATAC_anno**  \
   Annotation of scATAC-seq peaks with transcription factor motifs.  \
\
8. **8_GRN_clustering**  \
   Clustering of intra-module gene regulatory networks (GRNs).  \
\
9. **9_GRN_modules**  \
   Example scripts for GRN module construction and processing.\
\
10. **10_CIT**  \
    Causal inference integration (CIT) model.  \
